# Livebook Companions

These notebooks are the interactive companions for the `horizon_engine` series.

Each notebook uses a local path dependency back to its lesson directory, so the code you run in Livebook stays tied to the lesson implementation in this repository.

## Setup

If you do not already have Livebook installed:

```bash
mix escript.install hex livebook
```

Then start Livebook from the repo root:

```bash
livebook server livebooks
```
